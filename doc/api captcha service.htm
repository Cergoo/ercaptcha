2.
assign
check the entered captcha
request get
http://api.ercaptcha.net/captcha/check/'id'/'value of captcha'
response
'0' => 'invalid query format',
'1' => 'ok',
'2' => 'code of protection against robots entered correctly, repeat the entry',
'3' => 'code of protection against robots outdated, repeat the entry',
'4' => 'attempt to re-use the captcha',
addition
code 0 is returned:
- in case of technical problems (incorrect id captcha).
code 1 is returned:
- if successful, the recognition of previously not recognized captcha.
code 2 is returned:
- in case of unsuccessful recognition captchas.
code 3 returns:
- sometimes when you try to captcha recognition when it was over 2 hours from the time of its generation;
- always when you try to recognition captcha after more than 4 hours from the time of its generation.
code 4 is returned:
- when you try to re-use previously captcha recognized;
- after 3 failed attempts to recognize one captcha.
